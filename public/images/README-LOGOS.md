# Institution Logos

This directory should contain the official logos for the affiliated institutions.

## Required Logo Files

1. **URV Logo**: `urv-logo.png`
   - Source: https://www.urv.cat/en/university/communication/institutional-identity/logos/
   - Recommended size: Height of 60-80px, maintain aspect ratio
   - Format: PNG with transparent background preferred

2. **TecnATox Logo**: `tecnatox-logo.png`
   - Source: https://www.tecnatox.urv.cat/en/
   - Recommended size: Height of 60-80px, maintain aspect ratio
   - Format: PNG with transparent background preferred

3. **IISPV Logo**: `iispv-logo.png`
   - Source: https://www.iispv.cat/en/
   - Recommended size: Height of 60-80px, maintain aspect ratio
   - Format: PNG with transparent background preferred

## Logo Usage Guidelines

- Ensure you have permission to use the logos
- Follow the official branding guidelines from each institution
- Maintain proper clear space around the logos
- Do not alter or modify the logos without permission
- Use high-resolution versions for web display

## Fallback

If logos are not available, the footer will display text fallbacks (URV, TecnATox, and IISPV) automatically.
